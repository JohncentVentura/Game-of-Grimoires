# GAME OF GRIMOIRES

* Horizontal & Vertical = Movement
* Fire1 = Use Card 1
* Fire2 = Use Card 2
* Fire3 = Use Card 3
* Fire4 = Use Card 4
* Fire5 = Normal Attack
* Fire6 = Utility Move

Project Settings > Physics 2D
* Gravity > Y = 0
